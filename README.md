# Pulse-Compression
compress.py contains the signal class 
- all the code needed to run a variety of compression and noise reduction techniques on input data, and for calculating performance metrics
- includes methods for plotting 1D and 2D results
- includes a method for running a predefined selection of filters and plotting the results

matchfilter_examples.py is used to run various instances of the signal class, and obtain all the main results

analysis.py is used to visualise the performance metric results
